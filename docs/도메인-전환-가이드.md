## 핵심 한 줄

**doregi.com에서 `drnews.co.kr`의 네임서버를 Cloudflare 네임서버로 변경한 뒤, Cloudflare Pages에 커스텀 도메인으로 연결한다.**

---

## 전환 절차

### 1단계 — Cloudflare에 도메인 추가
1. Cloudflare 대시보드 로그인 → **Add a Site**
2. `drnews.co.kr` 입력 → Free 플랜 선택
3. Cloudflare가 기존 DNS 레코드를 스캔 → **전용 네임서버 2개**를 안내
   - 예: `aaa.ns.cloudflare.com`, `bbb.ns.cloudflare.com` (계정마다 다름)
4. 이 네임서버 2개를 메모

### 2단계 — doregi.com에서 네임서버 변경
1. **doregi.com 로그인** (등록자 계정)
2. 도메인 관리 → `drnews.co.kr` → **네임서버 변경**
3. 기존 `ns1.cafe24.com` 등 **전부 삭제**
4. 1단계에서 받은 **Cloudflare 네임서버 2개 입력** 후 저장
5. (.kr 도메인은 네임서버 최소 2개 필요)

### 3단계 — Cloudflare에서 전환 확인
- Cloudflare 대시보드에서 도메인 상태가 **"Active"** 로 바뀔 때까지 대기
- 네임서버 전파: 보통 수십 분 ~ 최대 **24~48시간**

### 4단계 — Pages 커스텀 도메인 연결
1. Cloudflare → **Pages → renewaldrnews → Custom domains**
2. **`drnews.co.kr`** 추가
3. **`www.drnews.co.kr`** 추가 (선택, 권장)
4. SSL/TLS 인증서 자동 발급 (수 분 소요)

### 5단계 — www 리다이렉트 (선택)
- `www.drnews.co.kr` → `drnews.co.kr` (또는 반대)로 통일
- Cloudflare Rules 또는 Pages 도메인 설정에서 처리

### 6단계 — 마무리
- `https://drnews.co.kr` 접속 → 리뉴얼 사이트 정상 표시 확인
- 카페24 **웹호스팅 해지** 가능 (도메인 등록은 도레지에서 유지)

## 코드 측 (이미 준비됨 ✅)

리뉴얼 코드의 SEO 메타·사이트맵·OG·robots가 **이미 `https://drnews.co.kr` 기준**으로 작성되어 있어 도메인 연결 시 그대로 정상 동작한다. (별도 코드 변경 불필요)

- `app/layout.js` — `metadataBase: https://drnews.co.kr`
- `app/sitemap.js`, `app/robots.js`, `app/news-sitemap.xml` — drnews.co.kr 기준

## 주의사항

1. **전파 시차** — 네임서버 변경 후 전 세계 전파까지 시간차. 그 사이 일부 사용자는 구/신 사이트가 섞여 보일 수 있음
2. **카페24 호스팅 유지 타이밍** — 전환 완료 확인 전까지 카페24 서버는 켜둘 것 (롤백 대비)
3. **메일(MX) 레코드** — `@drnews.co.kr` 이메일을 쓰고 있다면, 네임서버 이전 시 Cloudflare DNS에 **MX 레코드를 반드시 다시 등록**해야 메일이 끊기지 않음 (1단계 스캔에서 누락될 수 있으니 별도 확인)
4. **기존 DNS 레코드** — 카페24에 설정된 서브도메인/기타 레코드가 있으면 Cloudflare DNS에 옮겨 적기
